# 智能社区停车场管理系统 java757

## 项目概述


## 技术栈

### 开发环境

- JDK 21
- MySQL 8

### 前端技术

- Vue 2
- Element UI
- ECharts
- BaiduMap

### 后端技术

- Spring Boot
- Spring MVC
- MyBatis Plus
- Swagger 3

### 第三方服务

- 腾讯OCR文字识别接口
- 百度地图接口
- ECharts 图表接口

## 功能模块

### 车场管理

- 物业公司管理
- 停车场管理
- 车辆管理
- 停车场地图

### 停车管理

- 车牌识别
- 停车记录
- 缴费记录

### 系统管理

- 用户管理
- 角色管理
- 菜单管理
- 日志管理

## 系统角色

系统包含以下角色：

- 物业管理员
- 停车场管理员
- 停车用户

## 运行环境

- 前端：现代浏览器
- 后端：Java运行环境（JDK 21+）

## 部署步骤

1. 环境搭建：确保JDK 21和MySQL 8已正确安装和配置。
2. 数据库部署：执行数据库初始化脚本，建立项目所需的数据表。
3. 后端部署：编译后端代码，运行Spring Boot应用。
4. 前端部署：编译前端代码，部署到Web服务器。

## 目录结构

```
project/
│
├── frontend/               # 前端代码
├── backend/                # 后端代码
└── scripts/                # 数据库初始化脚本
```

## 核心亮点

- 整合腾讯OCR技术，实现高效车牌识别。
- 借助百度地图接口，提供停车场位置信息。
- 使用ECharts进行数据可视化，助力决策分析。
- 采用Spring Boot框架，确保系统的高效稳定运行。
- 全面的系统管理功能，保障系统安全与高效运维。
